## 自身の回答

```python
input_line = int(input())
arr = input().rstrip().split(' ')
# print(arr)
for i in range(input_line):
    x = arr[i]
    j = i -1
    while j >= 0 and arr[j] > x:
        arr[j+1] = arr[j]
        j-=1

    arr[j+1] = x
    print(*arr)
```

改善点
・左端は整列済みと扱う
・文字列リストを数値リストに変換する

input_line = int(input())
