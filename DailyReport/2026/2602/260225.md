## 改善(AI)

```js
全員の計算結果がマイナスになっても『結果的に0が出力される』という構造になっていて、運良くテストケースは通るかもしれない。でも、もし『各参加者の点数をすべて出力しろ』という仕様に変わったらバグになる。コードの意図を明確にするためにも、個人の点数計算の最後に 0 で下限を設ける処理を入れたい
```

```js
在のJavaScriptの標準入力の受け取り方（readlineモジュールによる非同期処理）は汎用的ですが、Paizaのような競技プログラミング環境においては、fs.readFileSync('/dev/stdin', 'utf8') を用いて入力を一括で同期的に読み込む手法が『実行速度』および『記述のシンプルさ』の観点で定石とされています。
```

```js
変数宣言の最適化: 変更されない変数には const を使用し、意図しない再代入を防ぐ堅牢な設計にしました（song配列や、ループ内の hz, differ など）。
→constにしてループで値を取り直しているのはこれは再代入にはならないのは驚き
→結論から言うと、再代入にはならず、エラーにもならないんだ。なぜなら、for文の中のブロック（ { } で囲まれた部分 ）は、ループが1周するごとに『完全に新しいスコープ（変数の有効範囲）』を作り直しているから
→
``````js
// 【エラーになる例】ループの外で宣言した場合
const hz = 0; 
for (let j = 0; j < M; j++) {
    hz = Number(input[currentLine++]); // ← 同じ箱に2回目を入れようとするのでエラー！
}

// 【エラーにならない例】ループの中で宣言した場合（今回のパターン）
for (let j = 0; j < M; j++) {
    const hz = Number(input[currentLine++]); // ← 毎回新しい箱が作られるのでOK！
}
``````
このコードが実行される際、内部的には『1周目専用の hz』『2周目専用の hz』というように、毎回全く別の新しい箱（変数）が作られ、ループが1周終わるごとに破棄されています。
つまり、『同じ箱の中身を書き換えている（再代入）』のではなく、『毎回新しい使い捨ての箱を用意して、1回だけ値を入れている（初期化）』という処理が行われているため、const のルール（一度入れた値は変更できない）に違反しないのです。
```

```
const fs = require('fs');

function main() {
    // AIロボット推奨：標準入力を一括で読み込み、改行で分割（処理速度・可読性向上）
    const input = fs.readFileSync('/dev/stdin', 'utf8').trim().split('\n');
    if (input.length === 0 || input[0] === '') return;

    let currentLine = 0;
    const [N, M] = input[currentLine++].split(' ').map(Number);

    // 課題曲の配列
    const song = [];
    for (let i = 0; i < M; i++) {
        song.push(Number(input[currentLine++]));
    }

    let maxScore = 0;

    // 各参加者の採点
    for (let i = 0; i < N; i++) {
        let score = 100;
        
        for (let j = 0; j < M; j++) {
            const hz = Number(input[currentLine++]);
            const differ = Math.abs(hz - song[j]);

            // 減点ロジック（そのまま活かしています）
            if (differ <= 5) {
                continue;
            } else if (differ <= 10) {
                score -= 1;
            } else if (differ <= 20) {
                score -= 2;
            } else if (differ <= 30) {
                score -= 3;
            } else {
                score -= 5;
            }
        }
        
        // 後輩の指摘：0点を下回らないようにする仕様を明示的に表現
        score = Math.max(0, score);

        // 最高得点の更新
        if (score > maxScore) {
            maxScore = score;
        }
    }

    console.log(maxScore);
}

main();
