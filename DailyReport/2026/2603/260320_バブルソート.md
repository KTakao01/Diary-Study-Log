# 考察：各ソート手法の違い

- 選択ソートは場所を最初に固定して、探した最小値を置く方式。
- バブルソートは、比較範囲をループの制約で記述して、隣り合う要素を比較・交換する方式。
- 挿入ソートは、整列済みの隣の未整列の状態のデータを取り出して、整列済みの適切な位置に挿入する。

## 自身の回答(1stロジックだけ正解、pythonの記法わからず外部情報を参照、2nd正解)

```python
def bubble_sort(arr,_len):
    for i in range(len-1):
        for j in range(len-1,i,-1):
            if arr[j] < arr[j-1]:
                arr[j],arr[j-1] = arr[j-1],arr[j]
        print(*arr)


N = int(input())
arr = list(map(int,input().rstrip().split(" ")))
bubble_sort(arr,N)
```

## AI添削

- 変数名 len について
  Pythonには、リストの長さを測る len() という標準関数があります。変数名に len を使ってしまうと、その関数が上書きされて使えなくなってしまうので、慣習的に n や length、あるいは \_len などを使うことが多いです。

- 関数の引数
  Pythonのリストは「参照渡し」なので、関数内で arr を書き換えると呼び出し元の arr も書き換わります。これは意図通りなので問題ありません。

- 値渡し (Pass by Value): データのコピーを渡す。関数の中で中身を書き換えても、元のデータは変わらない。

- 参照渡し (Pass by Reference): データの住所を渡す。関数の中で中身を書き換えると、元のデータも書き換わる。

```python
def bubble_sort(arr,_len):
    for i in range(_len-1):
        for j in range(_len-1,i,-1):
            if arr[j] < arr[j-1]:
                arr[j],arr[j-1] = arr[j-1],arr[j]
        print(*arr)


N = int(input())
arr = list(map(int,input().rstrip().split(" ")))
bubble_sort(arr,N)
```
